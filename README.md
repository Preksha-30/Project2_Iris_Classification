# Iris Flower Classification Using Machine Learning

## Project Overview

This project implements a Machine Learning classification model using the Iris Dataset. The objective is to classify Iris flowers into three species based on sepal and petal measurements.

## Objectives

* Load and explore the Iris dataset
* Perform data preprocessing
* Visualize the dataset
* Train multiple machine learning models
* Compare model performance
* Select the best-performing model

## Tools and Libraries

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn

## Machine Learning Models Used

1. Logistic Regression
2. K-Nearest Neighbors (KNN)
3. Random Forest Classifier

## Evaluation Metrics

* Accuracy
* Precision
* Recall
* F1 Score
* Confusion Matrix

## Results

The trained models achieved excellent performance on the Iris dataset. Random Forest, Logistic Regression, and KNN produced very high accuracy and successfully classified the flower species.

## Dataset

Iris Dataset

## Conclusion

This project demonstrates the complete Machine Learning workflow including data loading, preprocessing, model training, evaluation, and comparison. The models achieved high accuracy due to the well-structured nature of the Iris dataset.
